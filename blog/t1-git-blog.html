What are the benefits of version control? Version Control allows developers to manage and maintain the code efficiently. When a team works on the project, each developer can work on their code separately and merge the changes into a master code. It also allows developers to go back to the code's previous version in case there are bugs in the current version.

How does git help you keep track of changes? Git command status shows the changes made locally and whether or not it's in sync with the remote repository. Git commit command completes the changes and Git push command push the changes to the remote repository from local one.

Why use GitHub to store your code? GitHub provides web-based graphical interface, desktop and mobile integration. This compares to Git which is a command-line tool. GitHub also provides access control and several collaboration features such as wikis, task management, bug tracking and feature request for every project.
