Every language has its expressions you type ad nauseam. For example, in JavaScript you type a lot of console.log()s and for loops. Wouldn't it be splendid if there was a simple key combination to splice those into your code? That's easy to do with snippets! Any number of lines of code are at the whim of a few keystrokes! Behold this gif animation.

Snippet for console.log().

All I had to do was press alt + c. What if we need an array of integers up to 100? I have one for for loops too, alt + f. Check it out!

Snippet for for loops.

You get the point. Now I'll show you how to make your own. If you're using Sublime Text, just go to Preferences > Key Bindings â€“ User.

This will open the Sublime keymap JSON file. This has an array of objects. Simply add an object for the snippet you want.

The contents property has the actual code you want to insert. You'll notice a couple of special variables in there. $SELECTION will place the selected text in its place. ${0} is where the cursor will end up.

These snippets will save you a few seconds each time you use them. It adds up quickly! Combining this technique with Completions and other workflow strategies can save you a few days each year! Stay posted for my next workflow trick: aliases.
